## Design System

### Header Specifications
- **Colors**: Dark blue (#003366) and white design
- **Font**: Arial
- **Layout**: Two-row responsive flexbox
- **Components**:
  - Top row: "XS" logo, main navigation links (Browse, Deals with blue "NEW" badge, Resources, Support)
  - Search bar (light gray)
  - User auth links (login/account)
  - "Request Quote" button (solid blue)
  - "Cart" button (bordered blue)
  - Bottom row: Filter dropdowns (Type, Category, Location/Shipping, Price/Rate, Rating, Font, Color)
  - Notification bell with orange dot (#ff6600)
  - "Reset Filters" button
- **Hover States**: Blue accent (#0066cc)
- **Features**: Responsive design, CSS variables, JavaScript handlers for navigation/search/filtering
